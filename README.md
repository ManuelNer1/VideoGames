# Pacman
Para poder hacer uso de lo que es este juego 
1. Se tiene que clonar el repositorio<br>
`git clone https://github.com/ManuelNer1/VideoGames.git`
2. Entrar a nuestra carpeta en la computadora<br>
`cd VideoGames`
3. Correr el video juego<br>
`python3 pacman.py`
## Contenido
Lo que se realizó en esta práctica, fue hacer uso de github esto para la creación de ramas para posteriormente realizar un merge, lo que fue la modificación del código de pacman, donde la primera versión cuenta con el código original, la segunda con el mismo codigo solo que se modificó mediante el uso de flake8, en la tercera versión se tiene el codigo ya con lo que es un nuevo tablero y en la versión final se cuenta con el juego pero esta vez los fantasmas son más rápidos.
